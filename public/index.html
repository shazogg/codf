Help :
The "New" button is to create new document and delete the actual document.
The "Copy join link" button is to copy in clipboard the collaborate link, share it with others so that they can join it.
The "Download" button is to download localy the document in ".txt" file.
The "Upload" button is to upload to server a ".txt" file.
The "About and help" button is to access to this page.
The "Return to editor" button is to return to the editor page.
